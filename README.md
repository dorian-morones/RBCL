### Install local dev

_________________

```
npm run install
```

### Run storybook locally

_________________

```
npm run storybook
```


### Deploy a new version of the documentation website

_________________

> All document implementations must be from the main branch

- build storybook-static
- deploy to gh-pages

```
npm run storybool:export

npm run deploy-storybook

```
